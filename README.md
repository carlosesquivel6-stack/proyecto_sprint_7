# proyecto_sprint_7

Este proyecto es para practicar el tema de herramientas de software.
Se realizaron dos gráficos usando Python, utilizando dataframe, y se desplegó la aplicación en Render.

## Aplicación en línea

[https://proyecto-sprint-7-ztsp.onrender.com](https://proyecto-sprint-7-ztsp.onrender.com)

## Descripción

- **app.py**: Aplicación Streamlit con histograma y gráfico de dispersión sobre el conjunto de datos de vehículos en venta en EE.UU.
- **notebooks/EDA.ipynb**: Análisis exploratorio inicial del dataset `vehicles_us.csv`.
- **vehicles_us.csv**: Dataset con ~51 500 anuncios de venta de autos.
- **requirements.txt**: Dependencias del proyecto.
